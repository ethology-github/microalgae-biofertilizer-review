# 微藻类生物肥文献综述 - 质量评估标准清单

## 1. 纳入标准 (Inclusion Criteria)

| 维度 | 标准 |
|------|------|
| 主题相关性 | 研究对象为微藻类生物肥（Microalgae Biofertilizer），包括藻类提取物、藻类制剂、藻类共生体等 |
| 语种 | 中英文文献 |
| 发表时间 | 2015年1月1日至今 |
| 文献类型 | 同行评审期刊论文、综述文章（Review） |
| 可及性 | 需获取全文或至少摘要；无摘要者需单独复核 |

---

## 2. 排除标准 (Exclusion Criteria)

| 优先级 | 排除类型 | 说明 |
|--------|----------|------|
| P1 | 非微藻主题 | 以大型藻类（海带、紫菜等）为主、细菌/真菌生物肥研究 |
| P1 | 纯工艺工程类 | 仅聚焦藻类培养工艺、反应器设计、光生物反应器优化，无农业应用数据 |
| P1 | 专利文献 | 发明专利、实用新型专利 |
| P1 | 会议摘要 | 会议论文集摘要、Extended Abstract、Proceeding Paper（无完整正文） |
| P2 | 无摘要 | 无法获取摘要且无全文的文献（需团队负责人复核决定） |
| P2 | 重复发表 | 同一研究被识别为重复发表（保留最新/最完整版本） |
| P3 | 严重偏题 | 藻类用于污水处理、燃料生产、制药等与农业施肥无关的场景 |

---

## 3. 质量评分体系 (Quality Scoring System, 0-20分)

### 3.1 方法学质量 (0-10分)

#### 3.1.1 对照组设计 (0-3分)
| 评分 | 描述 |
|------|------|
| 0分 | 无对照组，或仅有历史文献对照 |
| 1分 | 设有对照组但设计不严格（如对照组条件不明确） |
| 2分 | 有合理对照组，但样本配对不完善 |
| 3分 | 随机对照实验（RCB）或配对设计，组间可比性强 |

#### 3.1.2 样本量与重复 (0-3分)
| 评分 | 描述 |
|------|------|
| 0分 | 样本量极小（n<3）或未说明重复数 |
| 1分 | n=3-5，有技术重复但无生物学重复说明 |
| 2分 | n=6-10，有生物学重复（n≥3 per treatment） |
| 3分 | n>10，生物学重复明确，统计功效有讨论 |

#### 3.1.3 统计分析 (0-4分)
| 评分 | 描述 |
|------|------|
| 0分 | 无统计分析，仅提供描述性结果 |
| 1分 | 使用了统计检验但方法不明确或未报告P值 |
| 2分 | 正确使用统计方法（t检验、ANOVA等），P值明确报告 |
| 3分 | 统计方法适当，多重比较校正（如 Tukey、Bonferroni） |
| 4分 | 统计完备，包含效应量、置信区间、功效分析或等效性检验 |

### 3.2 期刊质量 (0-6分)

| 评分 | 期刊级别 |
|------|----------|
| 6分 | SCI Q1 期刊 |
| 5分 | SCI Q2 期刊 |
| 4分 | SCI Q3 期刊 |
| 3分 | SCI Q4 期刊 |
| 2分 | 中文核心期刊（北大核心）、EI 期刊 |
| 1分 | 其他同行评审期刊（非SCI、非核心） |

> 期刊分区以发表年份的 JCR 分区为准。中文学术期刊按北京大学《中文核心期刊要目》认定。

### 3.3 主题相关性 (0-4分)

| 评分 | 描述 |
|------|------|
| 4分 | **完全匹配**：专门研究微藻类生物肥在农业中的应用，包括肥效、土壤改良、植物生长促进等核心议题 |
| 3分 | **高度相关**：微藻与其他生物刺激素（biostimulant）联合研究，或藻类多用途研究中的农业应用部分数据充分 |
| 2分 | **中度相关**：微藻基础研究（代谢、营养成分分析），关联农业应用但数据有限 |
| 1分 | **弱相关**：提及微藻可能用于肥料，但无直接实验数据支撑 |

### 3.4 总分分级与决策

| 分值范围 | 决策 | 说明 |
|----------|------|------|
| **≥12分** | **优先纳入** | 直接进入数据提取阶段 |
| **8-11分** | **需复核** | 需两名评审者独立评分并讨论一致性 |
| **<8分** | **排除** | 不纳入综述，需记录排除理由 |

> **一致性要求**：复核分数差异≥3分时，由第三人（团队负责人）裁定。

## 4. 两阶段初筛流程 (Two-Phase Screening)

### 阶段一：标题+摘要筛选 (Title & Abstract Screening)

**筛选依据（快速判断）：**
- 是否涉及微藻（Chlorella, Spirulina, Scenedesmus, Nannochloropsis, etc.）
- 是否与植物生长/土壤/生物肥相关
- 是否为原始研究或综述
- 时间范围是否符合（2015至今）

**输出：**
- 纳入 → 阶段二
- 排除 → 记录排除理由（见第5节）
- 不确定 → 进入阶段二

### 阶段二：全文复核 (Full-Text Assessment)

**复核内容：**
1. 逐项核对纳入/排除标准
2. 完成质量评分（Section 3）
3. 提取关键信息：藻种、研究对象、实验设计、主要结论

**输出：**
- 纳入（≥8分）
- 排除（<8分或不符合纳入标准）
- 记录排除的详细理由

## 5. 去重规则 (Deduplication Rules)

### 5.1 识别重复文献的优先级

| 优先级 | 匹配依据 | 阈值 |
|--------|----------|------|
| P1 | DOI 完全匹配 | 精确匹配 |
| P2 | 英文标题相似度 | Levenshtein 相似度 ≥ 0.85 或 token 重叠率 ≥ 90% |
| P3 | 作者 + 年份 + 标题关键词组合 | 三者中有两者高度相似 |

### 5.2 重复处理原则

- **同一研究的多篇发表**：保留最新、最完整的版本；若研究内容互补，可同时保留并在数据提取时标注关联
- **中英文版本同一研究**：优先保留英文版本（如已检索到）
- **会议摘要扩展为期刊论文**：保留期刊论文，剔除会议摘要
- **系统自动去重**：使用 Zotero / Rayyan / Covidence 等工具初筛后人工复核

## 6. 分类维度 (Classification Framework)

### 6.1 按研究机制分类

| 类别 | 说明 |
|------|------|
| M1 | 固氮/解磷/解钾机制研究 |
| M2 | 植物激素（auxin, cytokinin, GA等）分泌研究 |
| M3 | 土壤微生物群落调控机制 |
| M4 | 抗氧化/抗逆性诱导机制 |
| M5 | 营养成分直接供给机制 |

### 6.2 按藻种分类

| 类别 | 代表藻种 |
|------|----------|
| A1 | 绿藻门（Chlorophyta）：Chlorella, Scenedesmus, Chlamydomonas |
| A2 | 蓝藻门（Cyanobacteria）：Spirulina (Arthrospira), Nostoc, Anabaena |
| A3 | 硅藻门（Bacillariophyta）：Nannochloropsis, Phaeodactylum |
| A4 | 混合藻类/商品制剂 |
| A5 | 其他微藻 |

### 6.3 按应用场景分类

| 类别 | 说明 |
|------|------|
| C1 | 大田作物（粮食作物：水稻、小麦、玉米等） |
| C2 | 园艺作物（蔬菜、水果、花卉） |
| C3 | 土壤改良（退化土壤、盐碱地、重金属污染土壤） |
| C4 | 设施农业/有机农业 |
| C5 | 种子处理/育苗 |

### 6.4 按研究类型分类

| 类别 | 说明 |
|------|------|
| R1 | 田间试验（Field trial） |
| R2 | 盆栽/温室实验（Pot/Greenhouse experiment） |
| R3 | 实验室组培/水培实验（Lab/Hydroponic） |
| R4 | 综述/Meta分析 |
| R5 | 机制研究（无直接应用数据） |
